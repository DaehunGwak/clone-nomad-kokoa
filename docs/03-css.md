# 03. CSS

## CSS basic

- selector
  - id: `#`
  - class: `.`
  - tag: plain text
- property
  - in selector

```css
selector (id, class, tag name) {
  property-name: value;
}
```

## Inline css vs External css

- inline: file 내에서 style 태그로 css 관리
- external file 로 관리하면 여러 파일에 재사용 가능하기 때문에 inline 은 권장하지 않음
